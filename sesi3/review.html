Workshop Programming

Workshop Intro To Programming

Ingin belajar jadi programmer? Jangan lewatkan kesempatan ini! Workshop ini akan membahas pengenalan dasar tentang pemrograman, seperti tipe data, control flow, operator dan fungsi. Di workshop ini, kamu akan diberikan penjelasan dasar seputar struktur data yang berguna untuk membuat program lebih efisien dan cepat, jalur pemograman yang bisa diambil seperti web/mobile, serta bahasa pemograman yang berbeda-beda. Di akhir workshop ini akan ada demo coding, jadi kamu bisa secara langsung mempraktekkan ilmunya! Pembicara kita, Prabu, akan menyediakan video-video kursus online dari Team Treehouse dan CodeSchool untuk berbagai macam bahasa pemograman. Bawalah hardisk external atau flashdisk yang cukup besar, semuanya gratis!

Workshop Front End Web Developer

Di workshop ini, kamu akan belajar web design dari web developer professional menggunakan HTML, CSS, JavaScript dan Bootstrap, sebuah skill-set esensial untuk membuat design visual website yang menarik, mempertajam komunikasi pemasaran, dan peran terkait teknologi lainnya. Dalam workshop ini anda akan belajar membuat website yang full-customizeable sesuai kebutuhan dan permintaan, dinamis dan mobile friendly. Dengan menggunakan HTML5, CSS3, Bootstrap Framework & JQuery, anda juga akan mempelajari elemen-elemen serta tips & trick pembuatan website langsung dari developer berpengalaman.
